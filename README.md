## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sipaldrop/NeuraKnightBot-Sipal.git
   cd NeuraKnightBot-Sipal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Configuration

1. **Create Configuration Files**:
   Copy the template files to create your active configuration.
   ```bash
   cp accounts_tmp.json accounts.json
   cp config_tmp.json config.json
   ```

2. **Setup Accounts (`accounts.json`)**:
   Edit `accounts.json` and add your formatted tokens and proxies.
   ```json
   [
     {
       "token": "eyJhbGciOi...",
       "proxy": "http://user:pass@ip:port"
     },
     {
       "token": "eyJhbGciOi...",
       "proxy": null
     }
   ]
   ```

   > **Note**: `proxy` is optional. Set to `null` if not using a proxy.
